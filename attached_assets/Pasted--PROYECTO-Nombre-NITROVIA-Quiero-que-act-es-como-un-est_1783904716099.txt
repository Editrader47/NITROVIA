# PROYECTO

Nombre: NITROVIA

Quiero que actúes como un estudio profesional de desarrollo de videojuegos.

No quiero ejemplos, tutoriales, prototipos mínimos ni código de demostración.

Quiero construir el juego real.

Debes tomar el rol de:

- Director Técnico
- Arquitecto de Software
- Diseñador de Juegos
- Programador Senior
- Diseñador UX/UI
- Diseñador 3D
- Especialista en Optimización
- QA

Tú eres responsable de construir el proyecto completo.

------------------------------------------------------------

OBJETIVO

Crear un videojuego 3D moderno de carreras urbanas llamado NITROVIA.

El juego debe ser divertido, muy accesible y orientado al público casual.

NO debe ser un simulador profesional.

Debe sentirse rápido, espectacular y adictivo.

Debe funcionar tanto en PC como en Android.

Debe estar diseñado desde el inicio para poder exportarse posteriormente a iOS.

------------------------------------------------------------

MOTOR

Utiliza el motor que consideres técnicamente más adecuado para este proyecto.

Si existe una mejor alternativa que Godot para este tipo de juego dentro de Replit, justifícala y utilízala.

No tomes decisiones pensando en un ejemplo académico.

Toma decisiones pensando en publicar un juego comercial.

------------------------------------------------------------

OBJETIVO FINAL

Construir el juego completo.

No construir un demo.

No construir un tutorial.

No construir un ejemplo.

El proyecto debe quedar preparado para crecer durante años.
